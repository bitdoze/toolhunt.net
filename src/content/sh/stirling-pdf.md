Stirling PDF is a local-first PDF toolbox that covers most daily document operations without sending files to third-party cloud processors.

## Key Features

- **Core PDF Operations**: Merge, split, rotate, and reorganize PDF files.
- **Conversion Utilities**: Convert between PDF and common office/image formats.
- **Compression and Optimization**: Reduce file size for sharing and storage.
- **Security and Protection Tools**: Add or remove passwords and apply document controls.
- **OCR and Advanced Actions**: Extend workflows beyond basic editing.
- **Self-Hosted Web UI**: Run all operations in your own environment.

## Why Choose Stirling PDF?

- You handle sensitive documents and need privacy.
- You want one internal tool instead of many online converters.
- You prefer predictable, self-hosted processing.

## Docker Deployment

Deploy Stirling PDF with Docker Compose and mount persistent storage if needed for workflow data.

## Getting Started

1. Deploy Stirling PDF on your server.
2. Access the web UI and test common operations.
3. Optionally expose behind reverse proxy + TLS.
4. Standardize document workflows for your team.
